# Type Changes

## General Notes


The Fairy-type has been added to the game.

 - All Pokémon which are canonically Fairy Type as of X and Y are Fairy type in Redux.
 - In the Complete version, there are several additional Fairy types, like Renegade Platinum.

All Pokémon that are Fairy-type in Gen VI onwards are also Fairy-type in Redux. This is true for the Complete and the Vanilla versions.

For the type changes below, they are only present in the 'Complete' version of the Redux. If you don't like these changes, use the 'Vanilla' version patch instead.
