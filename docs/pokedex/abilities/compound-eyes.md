# Compound Eyes

## :material-information: Effect

!!! info "Full Description"

    Has no overworld effect.

!!! tip "Quick Summary"

    Increases moves' accuracy to 1.3×.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The Pokémon’s accuracy is boosted.
