# Zen Mode

## :material-information: Effect

!!! info "Full Description"

    This Pokémon switches between Standard Mode and Zen Mode after each turn depending on its HP. Below 50% of its maximum HP, it switches to Zen Mode, and at 50% or above, it switches to Standard Mode. This Pokémon returns to Standard Mode upon leaving battle or losing this ability. This ability has no effect if this Pokémon is not a darmanitan.

!!! tip "Quick Summary"

    Changes darmanitan's form after each turn depending on its HP: Zen Mode below 50% max HP, and Standard Mode otherwise.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    Changes the Pokémon’s shape when HP is halved.
