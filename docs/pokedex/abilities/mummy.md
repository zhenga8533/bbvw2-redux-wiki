# Mummy

## :material-information: Effect

!!! info "Full Description"

    Whenever a contact move hits this Pokémon, the attacking Pokémon's ability changes to Mummy. multitype is unaffected. If a Pokémon with moxie knocks this Pokémon out, the former's ability will change without taking effect.

!!! tip "Quick Summary"

    Changes attacking Pokémon's abilities to Mummy on contact.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    Contact with this Pokémon spreads this Ability.
