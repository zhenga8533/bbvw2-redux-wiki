# Telepathy

## :material-information: Effect

!!! info "Full Description"

    This Pokémon does not take damage from friendly Pokémon's moves, including single-target moves aimed at it.

!!! tip "Quick Summary"

    Protects against friendly Pokémon's damaging moves.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    Anticipates an ally’s attack and dodges it.
