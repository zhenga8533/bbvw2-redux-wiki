# Poison Touch

## :material-information: Effect

!!! info "Full Description"

    This Pokémon's contact moves have a 30% chance of poisoning the target with each hit. This counts as an extra effect for the purposes of shield dust. This ability takes effect before mummy.

!!! tip "Quick Summary"

    Has a 30% chance of poisoning target Pokémon upon contact.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    May poison targets when a Pokémon makes contact.
