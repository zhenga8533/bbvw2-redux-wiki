# Imposter

## :material-information: Effect

!!! info "Full Description"

    This Pokémon transforms into a random opponent upon entering battle. This effect is identical to the move transform.

!!! tip "Quick Summary"

    Transforms upon entering battle.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    It transforms itself into the Pokémon it is facing.
