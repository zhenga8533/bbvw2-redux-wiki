# Color Change

## :material-information: Effect

!!! info "Full Description"

    Triggers on every hit of multiple-hit moves.

!!! tip "Quick Summary"

    Changes type to match when hit by a damaging move.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    Changes the Pokémon’s type to the foe’s move.
