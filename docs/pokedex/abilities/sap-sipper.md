# Sap Sipper

## :material-information: Effect

!!! info "Full Description"

    Whenever a grass-type move hits this Pokémon, its Attack rises one stage, negating any other effect on it.

!!! tip "Quick Summary"

    Absorbs grass moves, raising Attack one stage.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    Boosts Attack when hit by a Grass-type move.
