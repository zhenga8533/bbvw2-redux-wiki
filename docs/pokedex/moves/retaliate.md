## :material-information: Effect

!!! info "Description"

    Inflicts regular damage. If a friendly Pokémon fainted on the previous turn, this move has double power.

!!! tip "Quick Summary"

    Has double power if a friendly Pokémon fainted last turn.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The user gets revenge for a fainted ally. If an ally fainted in the previous turn, this attack’s damage increases.
