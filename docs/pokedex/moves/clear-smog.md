## :material-information: Effect

!!! info "Description"

    Inflicts regular damage. All of the target’s stat modifiers are reset to zero.

!!! tip "Quick Summary"

    Removes all of the target’s stat modifiers.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The user attacks by throwing a clump of special mud. All status changes are returned to normal.
