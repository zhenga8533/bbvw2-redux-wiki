## :material-information: Effect

!!! info "Description"

    The target will act next this turn, regardless of Speed or move priority. If the target has already acted this turn, this move will fail.

!!! tip "Quick Summary"

    Makes the target act next this turn.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The user helps the target and makes it use its move right after the user.
