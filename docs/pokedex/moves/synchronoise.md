## :material-information: Effect

!!! info "Description"

    Inflicts regular damage. Only Pokémon that share a type with the user will take damage from this move.

!!! tip "Quick Summary"

    Hits any Pokémon that shares a type with the user.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    Using an odd shock wave, the user inflicts damage on any Pokémon of the same type in the area around it.
