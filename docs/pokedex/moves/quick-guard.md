## :material-information: Effect

!!! info "Description"

    Moves with priority greater than 0 will not hit friendly Pokémon for the remainder of this turn. If the user is last to act this turn, this move will fail. This move cannot be selected by assist or metronome.

!!! tip "Quick Summary"

    Prevents any priority moves from hitting friendly Pokémon this turn.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The user protects itself and its allies from priority moves. If used in succession, its chance of failing rises.
