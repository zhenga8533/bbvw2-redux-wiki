## :material-information: Effect

!!! info "Description"

    User switches position on the field with the friendly Pokémon opposite it. If the user is in the middle position in a triple battle, or there are no other friendly Pokémon, this move will fail.

!!! tip "Quick Summary"

    User switches places with the friendly Pokémon opposite it.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The user teleports using a strange power and switches its place with one of its allies.
