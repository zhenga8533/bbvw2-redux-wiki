## :material-information: Effect

!!! info "Description"

    electric moves inflict half damage, regardless of target. If the user leaves the field, this effect ends.

!!! tip "Quick Summary"

    Halves all Electric-type damage.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The user covers itself with mud. It weakens Electric-type moves while the user is in the battle.
