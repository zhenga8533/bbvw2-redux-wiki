## :material-information: Effect

!!! info "Description"

    Inflicts regular damage. If a friendly Pokémon used fusion bolt earlier this turn, this move has double power.

!!! tip "Quick Summary"

    With Fusion Bolt, inflicts double damage.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The user brings down a giant flame. This attack does greater damage when influenced by an enormous thunderbolt.
