## :material-information: Effect

!!! info "Description"

    Transfers the user’s held item to the target. If the user has no held item, or the target already has a held item, this move will fail.

!!! tip "Quick Summary"

    Gives the user’s held item to the target.

## :material-book-open: In-Game Description

!!! quote "Black 2 & White 2"

    The user passes its held item to the target when the target isn’t holding an item.
